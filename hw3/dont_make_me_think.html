Goodwill

The user should enjoy navigating your site. Here are some examples of things that will affect a user's goodwill.

Increase
  • Well organized
  • Main tasks are obvious
  • Prices, fees clearly listed
  • Doing extra work so I don't have to
  • Put effort into it
  • Relevant FAQ
  • Printer friendly pages
  • Easy to recover from errors
  • When in doubt, apologize
Decrease
  • Hiding information
  • Not auto-formatting input data
  • Asking for personal info
  • Faux sincerity
  • Excessive marketing
  • Amature looks
