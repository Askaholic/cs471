Web Design Best Practices

Follow convention. Clarity trumps consistency. Clear visual hierarchy. Make it obvious what's clickable. Reduce the amount of noise on the page. use bulleted lists. Highlight key terms

Follow convention

Conventions exist because they work well for delivering content in a way that the user will understand. Not only will following them save you time spent re-inventing the wheel, but users will also be comfortable navigating your site because they are already familiar with the conventions as they've seen them on other sites.

Clarity trumps consistency

If you can make something significantly clearer by slightly decreasing the consistency of the site, do it.

Visual hierarchies

Use visual queues to convey meta-information about the content of the site. Related topics should be grouped, either by putting them close together, or even by enclosing them in the same box. Think of a newspaper and how articles, their titles, and related images are visually close together.

Headings

The relative size of elements should also be used to indicate their importance. It is crucial that heading sizes are easy to distinguish with a quick glance at the page.

Bulleted lists

Take advantage of bullet points. These are a great way of creating content that is easy to scan.

Make it obvious what's clickable

Much of a users time is spent searching for the next thing to click, so it's important to make it obvious what's clickable and what isn't. Clickable things should be marked with a different color or style which should also change when hovered over with the mouse.

Mobile
Keep in mind that a large amount of web traffic comes from mobile devices that don't have a mouse, so clickable items need to be identifiable without the ability to hover.

Goodwill

The user should enjoy navigating your site. Here are some examples of things that will affect a user's goodwill.

Increase
  • Well organized
  • Main tasks are obvious
  • Prices, fees clearly listed
  • Doing extra work so I don't have to
  • Put effort into it
  • Relevant FAQ
  • Printer friendly pages
  • Easy to recover from errors
  • When in doubt, apologize
Decrease
  • Hiding information
  • Not auto-formatting input data
  • Asking for personal info
  • Faux sincerity
  • Excessive marketing
  • Amature looks
